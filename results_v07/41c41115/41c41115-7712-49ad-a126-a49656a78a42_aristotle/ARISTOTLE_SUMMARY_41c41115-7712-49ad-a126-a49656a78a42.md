# Summary of changes
## Summary

I made significant progress on formalizing Erdős Problem 672 — the theorem that no three distinct positive cubes form an arithmetic progression (equivalently, x³ + y³ = 2z³ with x,y,z > 0 implies x = y).

### File: `RequestProject/Erdos672.lean`

**What was proved:**

1. **`div_z_of_div_xy`** — If d divides both x and y, and x³ + y³ = 2z³, then d divides z. (Fully proved using p-adic valuation arguments.)

2. **`both_odd_of_coprime`** — If gcd(x,y) = 1 and x³ + y³ = 2z³, then x and y are both odd. (Fully proved using parity analysis.)

3. **`no_three_cubes_in_ap`** (the main theorem) — Fully proved **modulo** `descent_step`. The proof uses strong induction on z:
   - If gcd(x,y) > 1: divide out the gcd to get a smaller z, apply induction.
   - If gcd(x,y) = 1 and x ≠ y: apply `descent_step` to get a strictly smaller nontrivial solution, contradicting the induction hypothesis.

**What remains (`sorry`):**

4. **`descent_step`** — The core descent lemma: any coprime solution with x ≠ y yields a strictly smaller nontrivial solution. This is the heart of Euler's classical proof and requires:
   - Substitution s = (x+y)/2, t = (x-y)/2 → s(s²+3t²) = z³
   - Factorization in the Eisenstein integers ℤ[ζ₃] (a PID, available in Mathlib via `IsCyclotomicExtension.Rat.three_pid`)
   - Parameterization via cube roots in ℤ[ζ₃] to produce a new equation β³ + γ³ = 2α³ with α < z

   The descent step requires building significant algebraic number theory infrastructure around the Eisenstein integers (unique factorization, coprimality analysis of cyclotomic elements, coefficient extraction from cube expansions). While the PID property of ℤ[ζ₃] is available in Mathlib (used in the FLT(3) proof), the specific factorization and descent argument needed here requires substantial new infrastructure that could not be completed in this session.
