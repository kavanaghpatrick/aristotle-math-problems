## Summary

### Discovery: The original theorem `Erdos672With 3 2` is **false**

The original statement claims that no product of 3 terms in arithmetic progression (with first term coprime to the common difference) is a perfect square. This is **false**.

**Counterexample:** n=18, d=7 gives the arithmetic progression 18, 25, 32 with gcd(18,7)=1, and 18 × 25 × 32 = 14400 = 120².

### What was proved

1. **`erdos_672_k3_l2_false : ¬ Erdos672With 3 2`** — A formal proof that the original statement is false, using the counterexample (n=18, d=7, y=120).

2. **`erdos_672_consecutive_k3`** — The *correct* classical result: for all n > 0, n(n+1)(n+2) is never a perfect square. This is the d=1 specialization (consecutive integers), which is what the Erdős-Selfridge theorem addresses for k=3.

The proof of the correct theorem uses the key insight that n(n+2) = (n+1)²-1, so if n(n+1)(n+2) = y², then since gcd(n+1, n(n+2)) = 1, both n+1 and n(n+2) must be perfect squares. But n(n+2) = (n+1)²-1 being a perfect square leads to a gap-between-consecutive-squares contradiction.
