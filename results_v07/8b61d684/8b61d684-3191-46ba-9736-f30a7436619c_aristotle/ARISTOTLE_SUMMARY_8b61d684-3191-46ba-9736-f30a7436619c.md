# Summary of changes
## Erdős Problem 985 — Formalization

**File**: `RequestProject/Erdos985.lean`

### What was done

I formalized Erdős Problem 985 in Lean 4 with Mathlib, which states:

> For every odd prime `p`, there exists a prime `q < p` that is a primitive root modulo `p` (i.e., `orderOf (q : ZMod p) = p - 1`).

The file contains:

1. **`exists_primitive_root_lt`** (proved ✅): A helper lemma establishing that every odd prime `p` has a primitive root `g` with `1 < g < p`. This uses the fact that `(ZMod p)ˣ` is cyclic (via `IsCyclic.exists_generator`) and the totient formula for primes.

2. **`erdos_985`** (sorry — open problem): The main theorem statement, left as `sorry` because this is a known **open conjecture** in number theory.

### Why the main theorem remains open

This conjecture is related to Artin's conjecture on primitive roots. The natural approach — taking a primitive root `g` and showing one of its prime factors must also be a primitive root — fails because the lcm of proper divisors of `p-1` can equal `p-1` itself (e.g., `lcm(4, 3) = 12` for `p = 13`). Heath-Brown (1986) proved that at least one of {2, 3, 5} is a primitive root for infinitely many primes, but the full conjecture remains unresolved. Multiple proof strategies were attempted with the theorem-proving subagent without success, consistent with the problem's open status.
