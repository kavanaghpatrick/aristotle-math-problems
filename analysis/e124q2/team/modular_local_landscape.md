## 1. The objects

`B_d := S(d,0)` = {sums of distinct powers d^j, j≥0} = base-d numbers with digits ∈ {0,1}.
`S(d,k) = d^k·B_d` (scaling lemma, sumset peer [[sumset_reduction_scaling]]).
Target sumset: `Σ(D,k) := ∑_{d∈D} d^k·B_d`. (Q) asks: is Σ(D,k) cofinite when all d≥3,
∑1/(d−1)≥1, gcd(D)=1, k≥1?

Local image: `Σ(D,k) mod m = ∑_{d∈D} (d^k mod m)·(B_d mod m)`, where `B_d mod m` is the additive
**submonoid** of ℤ/m generated by the residues {d^j mod m : j≥0}, with the rule that a residue in
the eventual cycle has unbounded multiplicity and a pre-period residue has multiplicity 1.

---

## 2. THEOREM A (coprime ⇒ full). gcd(d,m)=1 ⟹ B_d mod m = ℤ/m.

**Proof.** d a unit ⟹ j↦d^j mod m is purely periodic, so 1=d^0 lies in the cycle (unbounded
multiplicity). The additive submonoid generated by an unbounded-multiplicity copy of 1 is all of
ℤ/m. ∎ *(Verified: no counterexample, d∈[3,30), m∈[2,80).)*

## 3. THEOREM C (prime power, p|d). For p|d, v=v_p(d), j*=⌈a/v⌉:
`B_d mod p^a` = { subset sums of d^0,…,d^{j*−1} mod p^a }, exactly 2^{j*} residues — the "low
base-d digits" Cantor set. (d^j ≡0 mod p^a for j≥j*.) *(Verified for many (d,p,a).)*

## 4. THEOREM B′ (CRT coupling — the subtlety).
`B_d mod m` is generally a PROPER subset of ∏_{p^a‖m}(B_d mod p^a): the digit pattern (Finset of
included powers) is shared across all prime components of a single d. So `B_d mod m` = CRT image of
the *diagonal* subset-sum set of the vector sequence (d^j mod p_1^{a_1},…). The coupling is real
(fails to factor for d∈{6,10,12,14,…}) but only within one base.

## 5. CENTRAL LOCAL LEMMA (L) — gcd(D)=1 ⟹ full residue coverage, every m, every k≥1.

> For all m and all k≥1: if gcd(D)=1 then ∑_{d∈D} d^k·B_d ≡ ℤ/m.

**Proof (CORRECTED to Aristotle's subgroup argument — canonical; see note below).** Fix m. Let
H_d := (d^k·B_d) mod m = the achievable residues from base d. Each H_d is the additive **submonoid**
of (ℤ/m, +) generated by {d^j mod m : j ≥ k}. The full sumset's residue set is
H := ∑_{d∈D} H_d (Minkowski sum of submonoids), again a submonoid of ℤ/m containing 0.

Step 1 — **H is a subgroup.** A submonoid of a *finite* additive group is a subgroup (for any x,
the cyclic submonoid ⟨x⟩ is finite, so some n·x = 0, whence −x = (n−1)·x ∈ ⟨x⟩ ⊆ H). [Aristotle:
`neg_mem_of_finite`.] So H ≤ ℤ/m is a subgroup of the cyclic group ℤ/m.

Step 2 — **per-prime witnesses.** Since gcd(D)=1, for each prime p | m there is d_p ∈ D with
p ∤ d_p. Then d_p is a unit mod p (indeed mod the p-part), and 1 = d_p^0 is in the *cycle* of
j ↦ d_p^j (Theorem A), so a suitable power d_p^{j} ≡ (a unit) is achievable; concretely H contains
an element not divisible by p (take base d_p's contribution = d_p^{k+t} for t making the exponent
land on a unit residue; d_p^k itself is a unit mod p, so d_p^k mod m is in H and is ≢ 0 mod p).
[Aristotle: `per_prime_witness`.]

Step 3 — **a subgroup of cyclic ℤ/m hitting a non-multiple of every prime p|m is everything.** A
subgroup of ℤ/m is dℤ/m for some d | m. If d > 1, pick a prime p | d; then every element of dℤ/m
is divisible by p, contradicting the witness of Step 2 (which is ≢ 0 mod p). Hence d = 1 and
H = ℤ/m. [Aristotle: `submonoid_top_of_witness`.] ∎

**This REPLACES my original Step-3 argument, which was FLAWED** (Aristotle caught it): I had tried a
per-block CRT recombination "assign each prime to a base d_p, force d_p's pattern to hit its block
and vanish elsewhere." That cannot be forced — a single base d contributes ONE number mod m, which
generally *leaks* into the other CRT blocks; the per-block bookkeeping is not available. The correct
argument is global (H is a subgroup; per-prime witnesses force it to be all of ℤ/m) and needs no CRT
block-assignment. The CONCLUSION was always right (triple-verified empirically), but the canonical
PROOF is Aristotle's. Formalization: `submissions/results_jun10/residue_coverage_x/…/RequestProject/Main.lean`

## 6. EFFECTIVE BOUND (quantitative form of (L)) — for the size-control / carry-debt analysis.

Let `Σ_T(D,k) := ∑_{d∈D}{subset sums of d^k,…,d^{k+T−1}}` (window of T powers per base).

**Bound.** For gcd(D)=1, `Σ_T(D,k) ≡ ℤ/m` as soon as `T ≥ m`. *(Verified: T=m always suffices,
D∈{(6,10,15),(3,4,7),(6,15,35),…}, k≤9, m<120 — zero failures.)*

**Sharpness — linear, NOT logarithmic, in the worst case.** If for some p^a‖m the *only* base
coprime to p is d_p with d_p ≡ −1 (mod p^a) (multiplicative order 2), then the powers alternate
{1,−1} and subset sums of T of them reach only an interval of length T, so coverage needs
`T = p^a − 1`. *(Verified exactly: d=m−1, m∈{4,8,16,32,64,128,9,27,81,25,49} ⟹ T=m−1.)*
Concretely D=(6,10,15), m=16, k=5: bases 6,10 vanish mod 16 (even), only 15≡−1 survives ⟹ T=15.

**Why this matters for (Q) — the genuine remaining difficulty.** Local coverage may *require*
powers up to position ~k+m, i.e. a summand as large as d^{k+m}. To represent a specific n EXACTLY
(all digits 0/1, summands ≥0 ⟹ each summand ≤ n), every used power must be ≤ n. So fixing the
residue of n modulo a slowly-covered prime power p^a can force a power d^{k+m} that may EXCEED n.
This is the **bounded-carry-debt** problem: local solvability with powers ≤ log_d(n). The linear
(not log) effective bound is exactly what makes this nontrivial — it is the precise point where
local theory hands off to the density argument (∑1/(d−1)≥1). Mapping for carry/sumset peers.
