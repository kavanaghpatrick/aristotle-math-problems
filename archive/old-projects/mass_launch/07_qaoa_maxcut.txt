## Problem Title: QAOA Depth-2 Optimality for MaxCut on 3-Regular Graphs (12 vertices)

---

## Executive Summary

Prove that QAOA (Quantum Approximate Optimization Algorithm) with depth p=2 achieves optimal approximation ratio for MaxCut on 3-regular graphs with 12 vertices. This has implications for near-term quantum algorithm performance.

---

## Background

### QAOA (Quantum Approximate Optimization Algorithm)

**Introduced:** Farhi et al. (2014)

**Goal:** Approximate solution to combinatorial optimization (MaxCut, Max-SAT, etc.)

**Parameters:**
- Depth p (number of alternating unitaries)
- Angles γ = (γ₁, ..., γₚ) and β = (β₁, ..., βₚ)

**Circuit:**
```
|ψ(γ,β)⟩ = U_B(βₚ) U_C(γₚ) ... U_B(β₁) U_C(γ₁) |+⟩^⊗n
```
where U_C is problem Hamiltonian and U_B is mixer.

### MaxCut Problem

**Input:** Graph G = (V, E)
**Goal:** Partition V into S, T to maximize |{(u,v) ∈ E : u ∈ S, v ∈ T}|

**Approximation Ratio:** QAOA_value / OPT

**Known:** For depth p=1, approximation ratio ≈ 0.6924 on 3-regular graphs.

---

## Concrete Parameters

**Graph:** 3-regular (each vertex has degree 3)
**Vertices:** n = 12
**Depth:** p = 2
**Target:** Prove QAOA with depth 2 is OPTIMAL for this class

## Goal 1: QAOA Approximation Ratio Lower Bound

### Task
Prove that QAOA with depth p=2 achieves approximation ratio ≥ α for MaxCut on 3-regular 12-vertex graphs.

### Approach: Explicit Optimization

**Strategy:**
1. Fix specific 3-regular graph on 12 vertices (e.g., Petersen-like)
2. Compute optimal angles (γ₁, γ₂, β₁, β₂) for QAOA
3. Evaluate expected value ⟨ψ(γ,β)|H_C|ψ(γ,β)⟩
4. Compare to OPT (maximum cut value)
5. Compute ratio α = QAOA_value / OPT

**For 12 vertices:** Computational optimization feasible.

### Formalization

```lean
-- Graph representation
structure Graph (n : ℕ) where
  edges : Finset (Fin n × Fin n)
  symmetric : ∀ {u v}, (u, v) ∈ edges → (v, u) ∈ edges

-- 3-regular property
def is_3_regular {n : ℕ} (G : Graph n) : Prop :=
  ∀ v : Fin n, (G.edges.filter (fun e => e.1 = v ∨ e.2 = v)).card = 3

-- MaxCut value
def maxcut_value {n : ℕ} (G : Graph n) (S : Finset (Fin n)) : ℕ :=
  (G.edges.filter (fun e => (e.1 ∈ S ∧ e.2 ∉ S) ∨ (e.1 ∉ S ∧ e.2 ∈ S))).card

  (Finset.univ.powerset.map (maxcut_value G)).max.getD 0

-- QAOA expected value
```

## Goal 2: Classical Algorithm Upper Bound

### Task
Prove that no classical randomized algorithm can achieve better approximation ratio than QAOA depth-2 in polynomial time (assuming hardness).

### Approach: Hardness Reduction

**Known:** MaxCut is NP-hard, best polynomial-time classical is Goemans-Williamson (0.878).

**For 3-regular graphs:** Specialized algorithms may do better.

**Prove:** Classical ratio ≤ 0.75 for 3-regular 12-vertex graphs (matching QAOA).

**Note:** This is the "optimality" claim - QAOA depth-2 matches best classical.
