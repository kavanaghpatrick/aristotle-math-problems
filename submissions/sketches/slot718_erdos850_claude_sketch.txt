OPEN GAP: Erdős 850 — twin same-radical triples (sketch-first pilot)
Source: FormalConjectures/ErdosProblems/850.lean
Domain: number theory
Pilot: 4×3 experiment, arm B (sketch_first), problem 1 of 1

Do there exist distinct positive integers x, y such that the three shifted pairs
(x,y), (x+1,y+1), (x+2,y+2) each share the same set of prime factors?

INFORMAL SKETCH (Claude Opus 4.7, ~350 words)

The naive heuristic underestimates how restrictive this is. Asking rad(x)=rad(y)
already constrains y to lie on a thin sublattice (the multiplicative group
generated by primes(x)). Imposing the SAME pairing on the shifts x+1↔y+1 and
x+2↔y+2 means we're intersecting three such sublattices that have NO obvious
common structure: primes(x), primes(x+1), primes(x+2) are arithmetically
independent for generic x.

A productive angle is to factor through the abc conjecture, but carefully —
the abc bound gives rad(x(x+1)(x+2)) ≫ x^{1−ε}, which says the *combined*
radical is large; it does NOT directly bound how often two such triples can
collide. The right reduction is to S-unit equations: fix S = primes(x) ∪
primes(x+1) ∪ primes(x+2); both x and y are S-integers with x+1, y+1 also
S-integers and x+2, y+2 S-integers. The pair of two-variable S-unit equations
x+1 ≡ y+1 (mod radical structure) and x+2 ≡ y+2 has only finitely many
solutions by Evertse, conditional on |S| being bounded.

The hardest case is when |S| is allowed to grow with x. Here a probabilistic
heuristic (Erdős–Pomerance style) suggests density 0 for solutions but does not
prove finiteness. The Baker-bound argument on linear forms in logs gives
x ≤ exp(C · |S|), which is not finitely many overall — only finitely many
per S. Sieving over S sizes 1, 2, 3, ... is the natural strategy.

Numerical evidence (Granville–Tucker tables, OEIS A006549 derivatives) finds
NO solution with x, y ≤ 10⁸. Status: genuinely open, no impossibility known,
no heuristic resolution either.

CANDIDATE LEMMAS (3–5, English only — no Lean tactics):

1. (S-restricted density) For any fixed finite prime set S, the count of
   x ≤ N with rad(x) | ∏S is O((log N)^|S|), uniformly in N. This rules out
   solutions of bounded |S| beyond a computable threshold.

2. (Three-way S-unit Evertse) For fixed S, the system { rad(x)=rad(y),
   rad(x+1)=rad(y+1), rad(x+2)=rad(y+2) } with x, y, x+1, y+1, x+2, y+2 all
   S-integers has at most C(|S|) solution pairs, where C is the Evertse
   constant for the three coupled S-unit equations.

3. (abc-conditional bound) Under abc with effective constant K, any solution
   with min(x,y) > K · max{primes(x(x+1)(x+2))} forces a divisibility
   contradiction in the product rad(x)·rad(x+1)·rad(x+2).

4. (Computational ceiling) Direct verification that no solution exists for
   x, y ≤ 10¹² using smooth-number enumeration tables. Resolves the existence
   side for the bounded range and forces any solution to require |S| ≥ 8.

5. (Equivalence reduction) The problem is equivalent to asking whether the
   diagonal map ℕ → ℕ³ via x ↦ (rad(x), rad(x+1), rad(x+2)) has a non-injective
   restriction to positive integers; i.e., whether this triple-radical fingerprint
   is injective on ℕ⁺.

theorem erdos_850 :
    (∃ x y : ℕ, 0 < x ∧ 0 < y ∧ x ≠ y
    ∧ x.primeFactors = y.primeFactors
    ∧ (x+1).primeFactors = (y+1).primeFactors
    ∧ (x+2).primeFactors = (y+2).primeFactors)
    ∨ ¬ ∃ x y : ℕ, 0 < x ∧ 0 < y ∧ x ≠ y
    ∧ x.primeFactors = y.primeFactors
    ∧ (x+1).primeFactors = (y+1).primeFactors
    ∧ (x+2).primeFactors = (y+2).primeFactors := by sorry
