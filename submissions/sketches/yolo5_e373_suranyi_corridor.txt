OPEN GAP: Erdős 373 (Surányi) — only solution to n! = a!·b! is (10, 7, 6)
Source: formal-conjectures/FormalConjectures/ErdosProblems/373.lean
Domain: nt

Surányi conjectured (1960s, open since) that the equation n! = a!·b!
with 1 < b ≤ a and a + 1 ≠ n has exactly one solution: (n, a, b) = (10, 7, 6),
i.e. 10! = 7!·6!. Numerical scan a ∈ [3, 200] confirms uniqueness; the only
prime-gap survivors in the corridor a + 2 ≤ n ≤ 2a - 1 are a finite explicit
set, all of which fail the factorial-product test except (a, n) = (7, 10).

Status: OPEN since 1960s. (10, 7, 6) is the only known non-trivial solution
and is conjectured unique. Prior submission slot 1260 (uuid 67d9e1d9) proved
the two corridor bounds n < 2a and a + 2 ≤ n sorry-free; slot 1288 attempted
the full closure via prime-gap survivor enumeration. No proof of the full
uniqueness exists in the literature.
