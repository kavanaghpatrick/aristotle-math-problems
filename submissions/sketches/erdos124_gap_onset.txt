OPEN GAP: Erdős 124 — effective onset of the interval-filling gap-condition (strict excess)

Domain: nt / combinatorics

For a finite set D of integers all ≥ 3 with β := ∑_{d∈D} 1/(d−1) and any k ≥ 1, consider the
merged atom set A = {d^j : d ∈ D, j ≥ k} and, for X ≥ 0, the partial sum S(X) = ∑ of atoms ≤ X.
Let C' := ∑_{d∈D} d^k/(d−1). The structural bound is S(X) ≥ m·β − C', where m is the smallest
atom exceeding X. Consequently, when σ := β − 1 > 0, the interval-filling gap-condition
(next atom ≤ S(X) + 1) holds for every atom m ≥ (C'−1)/σ.

Status: OPEN component of the BEGL96 problem (a NEW partial result, not in the literature). The
bound S(X) ≥ m·β − C' is exact and elementary. It yields a closed-form, transcendence-free onset
m₀ = (C'−1)/σ beyond which the Brown 1961 / Erdős 1962 interval-filling criterion holds. At σ = 0
