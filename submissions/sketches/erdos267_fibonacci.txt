OPEN GAP: Erdos 267 — Fibonacci reciprocal sums irrational
Source: FormalConjectures/ErdosProblems/267.lean
Domain: nt

Let n_1 < n_2 < ... with n_{k+1}/n_k >= c > 1.
Must sum 1/F_{n_k} be irrational?

theorem erdos_267 :
    ∀ (n : ℕ → ℕ) (c : ℚ), c > 1 → StrictMono n →
    (∀ k, c ≤ n (k+1) / n k) →
    Irrational (∑' k, 1 / (Nat.fib (n k) : ℝ)) := by sorry
