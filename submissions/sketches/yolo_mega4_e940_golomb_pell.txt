OPEN GAP: Infinitely many consecutive powerful number pairs (Mathlib novel)
Source: FormalConjecturesForMathlib/Data/Nat/Full.lean (Nat.Powerful = (2).Full); related to E364, E938, E940 in FormalConjectures/ErdosProblems/.
Domain: nt

A natural number n is powerful iff for every prime p dividing n, p^2 also
divides n. The conjecture below asserts that for every threshold N there
exists a powerful n > N with n+1 also powerful — equivalent to the set
{n : ℕ | Powerful n ∧ Powerful (n+1)} being infinite. The statement is phrased
per-N to remain structurally decidable; the Set.Infinite form is included.

theorem powerful_consecutive_pairs_unbounded (N : ℕ) :
    ∃ n, N < n ∧ Nat.Powerful' n ∧ Nat.Powerful' (n + 1) := sorry

theorem powerful_consecutive_pairs_infinite :
    {n : ℕ | Nat.Powerful' n ∧ Nat.Powerful' (n + 1)}.Infinite := sorry
