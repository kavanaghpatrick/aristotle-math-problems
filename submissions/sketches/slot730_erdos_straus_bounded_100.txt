OPEN GAP: Erdős-Straus conjecture — bounded verification n ∈ [2, 100]
Source: classical (Erdős-Straus 1948)
Domain: number theory

For every n with 2 ≤ n ≤ 100, the equation 4/n = 1/x + 1/y + 1/z has a
solution in positive integers x, y, z. Bounded version of Erdős-Straus.

theorem erdos_straus_bounded_100 :
    ∀ n : ℕ, 2 ≤ n → n ≤ 100 →
      ∃ x y z : ℕ, 0 < x ∧ 0 < y ∧ 0 < z ∧
        4 * x * y * z = n * y * z + n * x * z + n * x * y := by sorry
