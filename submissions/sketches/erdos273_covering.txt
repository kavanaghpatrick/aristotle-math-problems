OPEN GAP: Erdos 273 — Covering system with p-1 moduli (p>=5)
Source: FormalConjectures/ErdosProblems/273.lean
Domain: nt

Is there a covering system where every modulus has the
form p-1 for some prime p >= 5?

The p >= 3 version is known to be true (solved).

theorem erdos_273 :
    ∃ c : StrictCoveringSystem ℤ, ∀ i,
    ∃ p : ℕ, p.Prime ∧ 5 ≤ p ∧
    c.moduli i = Ideal.span {↑(p - 1)} := by sorry

Status: OPEN. The p>=3 case is solved (true).
The p>=5 constraint removes modulus 2 (from p=3).
