FEIT-THOMPSON p=3: QR CONTRADICTION FOR q ≡ 3 (mod 8)

This is a NOVEL proof resolving FT p=3 for all primes q ≡ 3 (mod 8).
Combined with the q ≡ 1 (mod 4) result (slot590), this covers 75%+ of primes.

SETUP: q prime, q > 9, q ≡ 2 (mod 3), A = q²+q+1 prime.
TARGET: Prove 3^q ≢ 1 (mod A) when q ≡ 3 (mod 8).

Note: q ≡ 2 (mod 3) and q ≡ 3 (mod 8) means q ≡ 11 (mod 24) for prime q > 3.

PROOF STRATEGY: Contradiction via Euler criterion and quadratic reciprocity.

=== PART 1: CONDITIONAL COMPUTATION (under assumption 3^q ≡ 1 mod A) ===

ASSUME FOR CONTRADICTION: 3^q ≡ 1 (mod A).

Known identities (already proven in slot590, slot598):
- q³ ≡ 1 (mod A), so q is a primitive cube root of unity mod A
- (1-q)² ≡ -3q (mod A)  [algebraic: (1-q)²=1-2q+q², and q²=-(q+1)]

Compute (1-q)^{2q} mod A:
  (1-q)^{2q} = ((1-q)²)^q = (-3q)^q = (-1)^q · 3^q · q^q
  Since q is odd: (-1)^q = -1
  By assumption: 3^q = 1
  Since q ≡ 2 (mod 3): q^q = q^{q mod 3} = q² = -(q+1)
  Therefore: (1-q)^{2q} = (-1) · 1 · (-(q+1)) = q+1

Compute higher powers:
  (1-q)^{4q} = (q+1)² = q²+2q+1 = -(q+1)+2q+1 = q
  (1-q)^{6q} = (1-q)^{4q} · (1-q)^{2q} = q · (q+1) = q²+q = -1
  (1-q)^{12q} = ((1-q)^{6q})² = (-1)² = 1

So (1-q)^{6q} = -1 and (1-q)^{12q} = 1, meaning ord(1-q) divides 12q
but does NOT divide 6q. For q > 9: ord(1-q) = 12q exactly.

Now compute (1-q)^{(A-1)/2}:
  A-1 = q(q+1), so (A-1)/2 = q(q+1)/2
  Need: q(q+1)/2 mod 12q = q · ((q+1)/2 mod 12)
  For q ≡ 3 (mod 8): q+1 ≡ 4 (mod 8), so (q+1)/2 ≡ 2 (mod 4)
  Also q ≡ 2 (mod 3), so q+1 ≡ 0 (mod 3), and (q+1)/2 ≡ 0 (mod 3)
  For q ≡ 11 (mod 24): (q+1)/2 ≡ 6 (mod 12)
  So q(q+1)/2 ≡ 6q (mod 12q)

  Therefore: (1-q)^{(A-1)/2} = (1-q)^{6q} = -1

CONCLUSION FROM ASSUMPTION: If 3^q ≡ 1 (mod A), then (1-q)^{(A-1)/2} ≡ -1 (mod A).

=== PART 2: UNCONDITIONAL LEGENDRE SYMBOL ===

By Euler's criterion, (1-q)^{(A-1)/2} ≡ (1-q / A) (mod A) where (· / A) is
the Legendre symbol. We now compute (1-q / A) unconditionally using QR.

Step 1: (1-q / A) = (-(q-1) / A) = (-1/A) · (q-1 / A)
  A ≡ 1 (mod 4) [since q² ≡ 1 mod 4, q ≡ 3 mod 4: A = 1+3+1 = 5 ≡ 1 mod 4]
  So (-1/A) = 1

Step 2: Write q-1 = 2m where m = (q-1)/2 is odd (since q ≡ 3 mod 4 → v₂(q-1)=1)
  (q-1 / A) = (2/A) · (m/A)

Step 3: Compute (2/A)
  For q ≡ 3 (mod 8): q² ≡ 1 mod 8, A = q²+q+1 ≡ 1+3+1 = 5 (mod 8)
  A ≡ 5 (mod 8) implies (2/A) = -1

Step 4: Compute (m/A) via Jacobi reciprocity
  Since (A-1)/2 = q(q+1)/2 is EVEN (q odd, (q+1)/2 even for q≡3 mod 4):
    (m/A) · (A/m) = (-1)^{(m-1)/2 · (A-1)/2} = 1  [since (A-1)/2 is even]
  So (m/A) = (A/m)_Jacobi

  Compute A mod m: q = 2m+1, so A = (2m+1)²+(2m+1)+1 = 4m²+6m+3
  A mod m = 3
  So (A/m) = (3/m)_Jacobi

Step 5: Compute (3/m) via Jacobi reciprocity
  m = (q-1)/2, q ≡ 11 (mod 24) → m ≡ 5 (mod 12)
  So m ≡ 2 (mod 3) and m ≡ 1 (mod 4)

  (3/m)(m/3) = (-1)^{(3-1)/2 · (m-1)/2} = (-1)^{(m-1)/2}
  m ≡ 1 (mod 4) → (m-1)/2 is even → (-1)^{(m-1)/2} = 1
  (m/3) = (m mod 3 / 3) = (2/3) = -1  [since 2 is a QNR mod 3]
  So (3/m) · (-1) = 1, giving (3/m) = -1

Step 6: Combine
  (m/A) = (3/m) = -1
  (q-1 / A) = (2/A) · (m/A) = (-1) · (-1) = 1
  (1-q / A) = (-1/A) · (q-1/A) = 1 · 1 = 1

CONCLUSION (UNCONDITIONAL): (1-q / A) = +1 for all q ≡ 3 (mod 8) with A prime.

=== PART 3: CONTRADICTION ===

By Euler's criterion: (1-q)^{(A-1)/2} ≡ (1-q / A) (mod A)
Part 1 gives (under assumption): (1-q)^{(A-1)/2} ≡ -1 (mod A)
Part 2 gives (unconditionally):  (1-q / A) = +1
So: -1 ≡ +1 (mod A), meaning A | 2. But A > 2. CONTRADICTION.

Therefore: 3^q ≢ 1 (mod A) for all prime q > 9 with q ≡ 3 (mod 8) and A prime.
